## üõí Order Management

### Viewing Orders
1. Navigate to **Order Management**
2. See orders from your team agents
3. Filter orders by status:
   - **All Orders**: All team orders
   - **Pending (Leader Review)**: Orders waiting for your approval
   - **Approved**: Orders you've approved
   - **Rejected**: Orders you've rejected

### Understanding Order Status
- **Agent Pending**: Order created by agent, waiting for your approval
- **Leader Approved**: You approved, waiting for admin approval
- **Admin Approved**: Admin approved, order is complete
- **Rejected**: Order was rejected

### Approving an Order (First Approval)
1. Go to **Order Management**
2. Filter by **"Pending (Leader Review)"**
3. Find the order you want to approve
4. Click on the order to view details
5. Review:
   - Client information
   - Order items and quantities
   - Payment information
   - Client signature
   - **Note**: You cannot see tax and discount (admin-only)
6. Click **"Approve"** button
7. Confirm in the dialog
8. Order status changes to "Leader Approved"
9. Order is sent to admin for final approval
10. Agent is notified

### Rejecting an Order
1. Go to **Order Management**
2. Find the order
3. Click on the order
4. Click **"Reject"** button
5. Enter rejection reason (optional but recommended)
6. Click **"Confirm Rejection"**
7. Order status changes to "Rejected"
8. Agent is notified

### Viewing Order Details
- Click on any order to see full details
- View all items, quantities, prices
- See payment proof (if uploaded)
- View client signature
- See order history and status changes

**Important**: You cannot see tax and discount fields - these are admin-only. You can see item prices and totals.

### My Orders
- Navigate to **My Orders** to see orders you've created (if you also act as an agent)
- Create new orders for your clients
- Track your personal order status

## üì¶ Inventory Management

### My Inventory
1. Navigate to **Inventory** ‚Üí **My Inventory**
2. See inventory allocated to you
3. View:
   - Product variants
   - Stock quantities
   - Allocated prices
   - Available stock (stock minus allocated to team)

### Teams Inventory
1. Navigate to **Inventory** ‚Üí **Teams Inventory**
2. See inventory for all leaders (including yourself)
3. View stock levels across all leaders
4. Monitor inventory distribution

### Understanding Available Stock
- **Available Stock** = Your Total Stock - Allocated to Team Members - Pending Orders (not yet approved)
- This is the stock you can allocate to agents
- Always check available stock before approving requests

### Pending Requests
1. Navigate to **Inventory** ‚Üí **Pending Requests**
2. See inventory requests from your team agents
3. Requests are grouped by agent and timestamp (bulk requests)

### Reviewing a Request
1. Go to **Pending Requests**
2. Find the request (grouped by agent)
3. Click **"Review Request"**
4. See all items in the bulk request:
   - Product name
   - Requested quantity
   - Your available stock
5. Review each item

### Approving a Request
1. Open the request review dialog
2. Review all items in the bulk request
3. Adjust approved quantities if needed (cannot exceed your available stock)
4. Add notes (optional)
5. Click **"Approve"**
6. Confirm in the dialog
7. Inventory is allocated to the agent
8. Your available stock is reduced
9. Agent is notified

### Forwarding a Request
1. Open the request review dialog
2. Click **"Forward"**
3. The request is forwarded to admin
4. You can add notes explaining why

### Denying a Request
1. Open the request review dialog
2. Click **"Deny"**
3. Enter a reason (optional)
4. Confirm the denial
5. Agent is notified

### Requesting Inventory from Admin
1. Navigate to **Inventory** ‚Üí **Request Inventory** (if available)
2. Select products you need
3. Enter quantities
4. Submit request
5. Wait for admin approval

### Team Remittances
1. Navigate to **Inventory** ‚Üí **Team Remittances**
2. See remittances from your team agents
3. View:
   - Agent who remitted
   - Items and quantities
   - Orders included
   - Remittance date
   - Digital signature

### Understanding Remittances
- Agents remit inventory back to you after sales
- Each remittance includes orders fulfilled
- Signatures confirm receipt
- Use this to track inventory returns from your team

## üìà Analytics & Targets

### Accessing Analytics
1. Navigate to **Analytics**
2. See analytics dashboard
3. View **Agent KPI** section (filtered to your team only)

### Agent KPI Section
- See performance metrics for your team agents
- View:
  - Orders (target vs actual)
  - Clients (target vs actual)
  - Revenue (target vs actual)
  - Quantity (target vs actual)
  - Achievement percentages

### Setting Agent Targets
1. Go to **Analytics**
2. Scroll to **Agent Key Performance Indicators**
3. Click **"Set Targets"** button (top right)
4. A dialog opens showing all your team agents
5. For each agent, set:
   - **Target Clients**: Number of clients for the month
   - **Target Revenue**: Revenue goal (‚Ç±)
   - **Target Quantity**: Quantity goal (units)
6. Enter values (use comma formatting for thousands)
7. Click **"Save"** for individual agent or **"Save All"** for all agents
8. Targets are saved for the current month
9. Achievement percentages are calculated automatically

### Understanding Targets
- Targets are set monthly
- System calculates achievement based on:
  - **Actual Clients**: Clients created this month
  - **Actual Revenue**: Revenue from approved orders this month
  - **Actual Quantity**: Total quantity from approved orders this month
- Achievement percentage = (Actual / Target) √ó 100%
- Displayed with color coding (green = good, yellow = needs improvement)

### Viewing Performance
- Monitor agent performance against targets
- Identify agents who need support
- Recognize top performers
- Use data for team coaching

**Note**: Only leaders can set targets. Admins can view agent KPIs but cannot set targets.

## ‚ö†Ô∏è Troubleshooting

### Cannot See Agent in Team
1. Verify agent is assigned to you
2. Check agent status (should be active)
3. Contact admin if agent is missing

### Cannot Approve Order
1. Check if order is in "Pending (Leader Review)" status
2. Verify you have permission
3. Check for error messages
4. Refresh the page

### Cannot See Available Stock
1. Check "Your Stock" column in Pending Requests
2. Available = Total Stock - Allocated - Pending Orders
3. Verify inventory is allocated to you
4. Check if requests are consuming available stock

### Request Not Showing
1. Check if request was already processed
2. Verify you're looking at Pending Requests page
3. Check filters
4. Refresh the page

### Cannot Set Targets
1. Verify you're on Analytics page
2. Check if "Set Targets" button is visible
3. Ensure you're viewing Agent KPI section
4. Contact admin if button is missing

### Agent Cannot Request Inventory
1. Verify agent is active
2. Check if agent has access to Request Inventory page
3. Verify you have available stock
4. Check system status
